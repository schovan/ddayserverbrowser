# ddayserverbrowser
D-Day Normady (Quake 2 mod) server browser. Requires Windows and [.NET Core 3.0 Desktop Runtime](https://dotnet.microsoft.com/download/dotnet-core/thank-you/runtime-desktop-3.0.1-windows-x64-installer).
